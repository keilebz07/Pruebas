oneM1 - Sitio web multipágina

Contenido incluido:
- index.html
- servicios.html
- nosotros.html
- portafolio.html
- contacto.html
- assets/css/styles.css
- assets/js/main.js
- assets/img/*

Características:
- Diseño responsive para móvil y escritorio
- Botón modo día/noche
- Footer en todas las páginas
- Logo actualizado
- Formulario configurado para enviar a onem1publicidad@gmail.com mediante FormSubmit
- Accesos a Facebook y WhatsApp
- Sección Nosotros separada en página propia
- Galería usando el material compartido

Importante:
- Bootstrap está integrado por CDN.
- El envío del formulario depende del servicio externo FormSubmit.
- Si más adelante quieres que el correo salga desde un hosting propio o API, se puede conectar a backend sin problema.

Publicación:
1. Descomprime el ZIP.
2. Sube todos los archivos a tu hosting.
3. Abre index.html para pruebas locales.
